## Prerequisites
- **Python 3.10+** (Make sure `python` is in your PATH)
- **Node.js 18+** (Make sure `npm` is in your PATH)

## Setup Instructions

### 1. Backend Setup (Django)
Open a terminal in the `backend/` directory.

```bash
# Create virtual environment
python -m venv venv
# Activate on Windows
.\venv\Scripts\Activate
# Activate on Mac/Linux
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# migrations
python manage.py makemigrations academics resources
python manage.py migrate

# Create Admin User
python manage.py createsuperuser

# Run Server
python manage.py runserver
```

The Backend API will run at `http://127.0.0.1:8000/`.

### 2. Frontend Setup (React + Vite)
Open a new terminal in the `frontend/` directory.

```bash
# Install dependencies
npm install

# Run Development Server
npm run dev
```

The Frontend will run at `http://localhost:5173/`.

## Features
- **Courses**: Browse courses by year.
- **Library**: Filter textbooks by subject.
- **Upload**: Submit materials (requires Backend to be running).
- **Admin**: Manage approvals at `http://127.0.0.1:8000/admin/`.
